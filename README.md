## Задание 1 v2

### Инструкция по сборке и запуску приложения
1. Установка зависимостей:

   ```
   pip install -r requirements.txt
   ```

2. Создание файлов для тестов:

   ```
   python create_files.py
   ```
   
3. Запуск сервера:

   ```
   python server.py
   ```
